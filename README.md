# Sakura : A interactive binary static analysing tool

## Screeshots
![avatar]

## Requirements
### python 2.x
### python libs:

* capstone
* pyelftools
* pwntools

`python2 -m pip install capstone pyelftools pwntools`

### Usage

`python main.py (filename)`

### Features

* disassemble function/ address/ section

* view relocation sections

* view sections
